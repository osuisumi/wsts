mylayer.open

调用方法：mylayerFn.open;

参数：

id:记录layer弹框独一无二的标志，可填数字，例如：(id: 999)

type:值为0~2。 “0”代表默认值，信息提示层； “1”页面层； “2”iframe层. 例如：(type: 1)

title:弹框的标题，默认为标题“提示”. 例如：(title: "新的弹层")

area:弹框的大小，第一个为弹层的宽；第二个为弹层的高度；不设置则为默认值'auto',自动获取宽高。 例如：(area: ['800','600'])

offset:弹框弹出的位置，第一个为弹层的top值，第二个为弹层的left值，不设置则为默认值'auto',自动居中。 例如：(offset: [400,200])

icon:type:0时设置的图标，0~6，不设置则默认为不显示图标。 例如：(offset: [400,200])

closeIcon:是否隐藏右上角的关闭图标，有true和false两个值，设置为false是隐藏关闭图标，默认为true.

shadeClose:点击遮罩层是否关闭弹层，设置为true，则点击遮罩层可关闭弹层，false反之。，默认为false。 例如：(shadeClose: true)

fix:弹层是否固定定位。默认为绝对定位，设置true时为固定定位。 例如：(fix: true)

zIndex:设置弹层的层级，不设置自动计算层级。 例如：(zIndex: 999999)

success:弹出成功后，执行函数。 例如：( success: function(){alert('成功了');} )

yes:弹出框里面的确定按钮。close: true,点击是否关闭，true为默认值，false为点击不执行关闭弹层。btnName: ,按钮class名，默认为'.mylayer-confirm'；yesFn : 点击后执行函数。

cancel:关闭弹出框后执行函数，默认为空

end:弹框关闭后，执行函数，默认为不执行
